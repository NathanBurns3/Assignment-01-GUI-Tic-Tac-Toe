Support files for Ass 01 GUI Tic Tac Toe from CP II
- SafeInput.java The console input library we created in CP I
- TicTacToe.java The complete working console version of TTT from CP I
- TicTacToeTile.java Defines a new type of JButton for TTT via sub-classing
- TicTacToeTileTester.java A naive example of how to us the TTT Tile with no GUI
- main.java Full GUI version of the Tester that shows how to create the Swing TTT Board
  using the tile.

  Your task in this assignment is to use the tile in the GUI to connect to the existing TTT console code
  instead of completely engineering a TTT game from scratch.
